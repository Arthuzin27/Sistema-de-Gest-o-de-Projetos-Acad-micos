# 📊 Sistema de Gestão de Projetos Acadêmicos

Aplicação web frontend desenvolvida para gerenciar projetos acadêmicos em um ambiente educacional, simulando um sistema completo de controle de dados e fluxo de informações.

O sistema foi construído utilizando **HTML, CSS e JavaScript puro (Vanilla JS)**, com persistência de dados no navegador através do **localStorage**, sem utilização de backend.

---

## 🚀 Funcionalidades

- Simulação de autenticação de usuários com controle básico de sessão
- Operações completas de CRUD (Create, Read, Update, Delete) para:
  - Cursos, Períodos Letivos e Turmas
  - Alunos
  - Professores e Coordenadores
  - Grupos de Projeto
  - Projetos
  - Estandes e Horários de Apresentação
- Relacionamento entre entidades (ex: turmas vinculadas a cursos, projetos vinculados a grupos)
- Validações básicas de dados nos formulários
- Busca e filtragem de informações
- Interface organizada com múltiplas telas de gerenciamento
- Persistência de dados utilizando localStorage

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage API

---

## 🔐 Autenticação

O sistema utiliza um modelo de **login simulado**, com usuários pré-definidos diretamente no código JavaScript, permitindo demonstrar conceitos de controle de acesso e gerenciamento de sessão no frontend.

> Não há autenticação real ou integração com backend.

---

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido com foco na prática de:

- Manipulação de DOM
- Tratamento de eventos em JavaScript
- Implementação de operações CRUD no frontend
- Gerenciamento de estado utilizando localStorage
- Organização e estruturação de código em aplicações web

---

## ⚠️ Observações

- Aplicação 100% frontend
- Dados armazenados localmente no navegador
- Não há persistência em banco de dados externo
- Projeto com fins educacionais
